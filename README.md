# 🐾 Прихисток — сайт для усиновлення котів

Односторінковий адаптивний сайт притулку для тварин. Дизайн імітує
"картку реєстрації" притулку: кожен кіт — це справа з ID, породою,
віком і нотатками куратора.

Чистий **HTML + CSS + JavaScript**, без збірщиків і залежностей.
Достатньо відкрити `index.html` у браузері.

## Структура проєкту

```
cat-shelter/
├── index.html   — розмітка трьох екранів (вхід, каталог, деталі кота)
├── style.css    — дизайн-токени, мобільна та десктопна версії
├── data.js      — масив котів (легко редагувати/додавати своїх)
├── app.js       — логіка: навігація, улюблені, тости
└── README.md
```

## Запуск локально

Варіант 1 — просто відкрити файл:

```bash
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

Варіант 2 — через VS Code:

1. Встановити розширення **Live Server**.
2. Правий клік на `index.html` → **Open with Live Server**.

Варіант 3 — будь-який локальний сервер:

```bash
python3 -m http.server 8000
# відкрити http://localhost:8000
```

## Адаптивність

Сайт має дві версії інтерфейсу, що перемикаються автоматично
залежно від ширини екрана (медіа-запит на **900px**):

| | Мобільна версія (< 900px) | Десктопна версія (≥ 900px) |
|---|---|---|
| Каркас | Рамка телефону на весь екран | Звичайна веб-сторінка на всю ширину |
| Вхід | Фото зверху, форма знизу | Спліт-екран: фото зліва, форма справа |
| Каталог | Список карток в один стовпчик | Сітка карток (grid, кілька в ряд) |
| Картка кота | Відкривається на весь екран | Виїжджає як бокова панель (drawer) справа, з затемненням фону |

Перевірити можна, просто змінюючи ширину вікна браузера —
переключення відбувається миттєво, без перезавантаження сторінки.

## Як додати свого кота

Відкрити `data.js` і додати новий об'єкт у масив `CATS`:

```js
{
  id: "226630",
  name: "Murchik",
  sex: "Самець",
  age: "2 роки",
  breed: "Сіамський мікс",
  color: "Білий з підпалом",
  weight: "4.2 кг",
  status: "У ПРИТУЛКУ",
  pawColor: "#2c5f8a",
  notes: "Опис вдачі та особливостей кота."
}
```

Картка та повна "справа" згенеруються автоматично — нічого більше
змінювати не потрібно.
